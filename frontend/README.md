# CodeContext frontend

Next.js App Router UI for the CodeContext monorepo.

See the [repository README](../README.md), [Roadmap](../docs/ROADMAP.md), and [Project Status](../docs/PROJECT_STATUS.md) for setup and progress.

**Local dev (from repo root):** `docker compose up --build` or `cd frontend && npm run dev` with the backend at `NEXT_PUBLIC_API_URL`.
